# Boilreact

[![Build Status](https://travis-ci.org/besarthoxhaj/boilreact.svg?branch=master)](https://travis-ci.org/besarthoxhaj/boilreact)

Simple boilerplate for react app.

### Production

```bash
  $ yarn install
  $ yarn build
  $ yarn start
```

Check `localhost:9000`.

### Development

In development run:

```bash
  $ yarn install
  $ yarn dev
```

Check `localhost:3000`.
